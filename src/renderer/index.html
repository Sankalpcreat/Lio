Setup Guide

Guide

    Voice Shortcut

    Toggle the live mic from anywhere

    Press once to connect and start the mic. Press again to stop listening. This build supports standard Electron accelerators; `Fn`-only shortcuts still need a native macOS helper.

    Active Shortcut
    Command + Shift + Space

    Current global toggle for connect + microphone.

    Change Shortcut

    Use Command, Control, Option, Shift, letters, numbers, function keys, or Space.

    For Fn shortcuts, also make sure macOS is not still using Globe/Fn for Dictation, Emoji, or another system action.
